This is a quick & dirty port of Christoph Nahr's quick & dirty port of Ian Nowland's extremely
useful CommentReflower plugin for Visual Studio 2003 and 2005.

https://github.com/kmunson/CommentReflowerVSIX/

Christoph Nahr's distribution: http://kynosarges.org/CommentReflower.html

Ian Nowland's distribution: http://commentreflower.sourceforge.net/

Deployment for Visual Studio 2012, 2013 and 2015: Simply run the VSIX installer to deploy.
